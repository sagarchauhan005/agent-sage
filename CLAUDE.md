# Sage

This repo is the Sage SDLC agent system. The root contract is [Agents.md](Agents.md), not this file.

Read Agents.md first, before doing any work here, even if no `/sage-*` command was typed. It defines the agent registry, workflow profiles, stack preferences, and global rules that apply to every phase agent in `agents/`.

If the task is plan/design/architect/build/qa/devops/release work, use the matching `/sage-*` skill in `.cursor/skills/` rather than improvising. See [README.md](README.md) for the full command list.
